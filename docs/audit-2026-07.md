# Full Project Audit — 2026-07-17

Two parallel Opus audits (engine/rules + UI/state/storage), findings independently verified by the team lead before acceptance. Baseline at audit time: lint clean, 123/123 tests, build green — every finding below was latent, not a regression.

Severity bar: **critical** = wrong computed numbers, data loss, or crash in realistic use · **major** = wrong in uncommon-but-real scenarios or contract violations · **minor** = robustness/doc drift.

## Findings

| # | Sev | Location | Defect | Status |
|---|-----|----------|--------|--------|
| 1 | CRITICAL | storage.js `normalizeCharacter` + no error boundary | Malformed imported JSON (e.g. `classLevels: "oops"`) persists, white-screens the app on render, auto-reselects on every reload; no UI recovery | **FIXED** — type-coercing normalize + ErrorBoundary with delete/reload |
| 2 | MAJOR | computeSheet CMD assembly | CMD ignored AC bonuses/penalties other than deflection+dodge; RAW says circumstance/insight/luck/morale/profane/sacred bonuses and **all** AC penalties apply (blinded, Rage, insight buffs gave wrong CMD) | **FIXED** — typed filter per RAW + tests |
| 3 | MAJOR | newCharacter.js | New characters start at `hpCurrent: 0` → Play tab shows 0/max in dying-red until manually healed | **FIXED** — default null (= start at max) + Full-heal button |
| 4 | MAJOR | buffLibrary.json enlarge-person | Buff carried −1 attack/−1 AC effects *and* told the user to set Size Large, double-counting the size shift | **FIXED** — effects removed, description reworded |
| 5 | MAJOR | mightClasses.json savant + skills.js | `"Knowledge (all except arcana)"` matched nothing → Savant got no Knowledge class skills | **FIXED** — enumerated the nine subskills |
| 6 | MAJOR | skills.json | Artistry and Lore (used by every Guile class's class-skill list) didn't exist as skills | **FIXED** — added as instantiable families |
